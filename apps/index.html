ZIPSite

VS Code inspired editor for web coding.

Open * Work in Progress

ZIPSIte is a VS Code style UI for web design using a virtual web server file system using a ZIP file. It uses javascript to simulate a working server and has Has minimal support for very basic Apache server with PHP.

Inlcudes

  • Minimal virtual PHP web server
  • Customized PHP template for multi-language sites
  • SVG editing using the text editor
  • SNS link preview for websites
  • Search functionality to search for text in the entire ZIP file
