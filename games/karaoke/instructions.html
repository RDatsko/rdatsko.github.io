Karaoke Box Instructions

The Karaoke Box uses standard MP3s with some custom media tags to allow for the files to be used as regular song files but still contain the data to play the game. This allows for easy customization without the need for additional files since everything is in one file.

Information

Currently, the easiest way to do this is to use MP3tag for Windows and make some changes. I plan on creating a web based editor later if I can find the resources to do so. I also plan to try to advance the editor to allow for a waveform view to allow for creating notes for your own custom songs. Also, I hope to try to do auto pitch detection if possible to give a starting point.
