COSMUIC

COSMUIC (pronounced "cosmic") is a CSS library that tries to minimize the use of loading sites with tons of classes and JavaScript. It's designed to act sort of like a template that you would experience in an office suite such as PowerPoint. It's designed to basically allow you to set things in the cosmuic.css file and then just add it to your page for site wide uniformity.

The goal is to create responsive websites and web applications using design principles such as the "60-30-10 Color Rule" and Material Design UI similar to Google while minimizing the need for classes or JavaScript and still having the ability to add some classes to enhance the appearance. This entire website was made using COSMUIC and shows how simple and quick it can be to use and customize.

Additionally, it adds things such as grids, cards, and hero sections with minimal need to fine code them. Just copy the code included from the samples from the necessary sections and make any adjustments as necessary. Each section is designed to provide as much information as possible to clearly identify options that are available.

It's easy to get started. Feel free to copy the code from the COSMUIC > Template link on the left to start with a simple template. Just make the changes as needed and then add the rest.

Note: This is work in progress and may change over time but will attempt to keep the existing structure to try to remove the need for modification to the existing HTML code. Also, becareful when using ::before and ::after psuedo elements as these are used on many of the elements for appearance.

Aditional Information:

Paragraphs by default are left aligned. However, if you wish to indent a paragraph, the class indent will indent the first line of the paragraph with with with of 2rem.

There are many people that have had multiple <br> tags, even me.

Well, no more need for <br><br><br><br>. Just use a <hr class="br{number}"> tag with a required {number} of lines. Want one blank line <hr class="br1">. Want two blank lines <hr class="br2">. There's support fo up to 10 lines. Each line is a height of the font in rem. If there is no class, then it will be a normal style <hr> tag.

Cards

Card styles are designated by a class named card followed by a style number. Cards are also designed to be placed inside a div.grid element. Cards also use a data-grid heml attribute to designate the number of sections the grid takes up. COSMUIC uses a 12 grid wide layout. refer to the grid section of the documentation for more information on the grid layout used by COSMUIC.

Card Stylization

There are typeds of card configurations built into COSMUIC.

  • This is the default for cards it uses the primary color as the backgrund color.
  • roundThis adds larger rounding to the card but otherwise is the same a style 1.
  • clearThe clear tag on this makes the background transparent and adds a border around the card.
  • clear roundThe clear tag on this makes the background transparent and adds a border around the card while adding larger rounding to the card.

Hero

Most websites now-a-days have a hero page of some sort. As many site use this, a hero section has been added to COSMUIC to make it simple.

Note: Currently this only supports right and full styles. It doe not currently support the vertical positioning of items yet.

Hero Stylization

The hero class make the section available as a hero. Due to the amount of configuration a few classes may need to be used. Be sure to always include hero before the other classes listed below.

  • fullThis keeps the entire hero section as one.
  • rightThis sections the hero section into two halfs with the image being the primary focus on the right.
  • leftThis sections the hero section into two halfs with the image being the primary focus on the left.

There are also additional options that pertain to both styles of the hero section however they differ slightly depending if it's a full width image or a half width image.

  • topSets the starting position to be the top of the hero section.
  • middleSets the starting position to be the middle of the hero section vertically.
  • bottomSets the starting position to be the bottom of the hero section.
  • center Sets the starting postiion to be centered horizontally, in either the right, left, or full hero parts. All elements will be centered. All centered elements default to auto wrap with a width of 50%.

Form Stylization Classes

There are four designs of forms. Use the class="..." to change the appearanec as listed below.

  •  No class makes it default with just the bottom border being displayed.
  • outlinedThis puts a line around the inputs and also the form legend.
  • filledThis fills the inputs with a lighter shade of the primary color.
  • outlined filledThis both fills the input and also puts an outline around them.
