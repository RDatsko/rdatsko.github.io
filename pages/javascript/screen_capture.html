Screen Recorder

To capture system audio on Mac OS X use the Chrome Tab option only. Trying to capture the anyother way other that this will not work on Mac OS X. It will only capture the microphone and the audio on that tab, not any other system sounds.

  1. Click Capture Screen
  2. Click Chrome Tab and select the tab you wish to record
  3. Make sure that Share tab audio is checked and click Share
  4. Click the Record Button
  5. Click the Stop button to stop recording
  6. Click the Download video file to download
Make sure that you have plenty of space before recording. The recording process is done on your computer and will be lost if you don't download the file after.
