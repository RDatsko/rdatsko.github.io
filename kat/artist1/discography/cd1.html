CDジャケットの背景をぼかしたのは、シンプルなコントラストをつけるためだ。 このサンプルで黒が選ばれたのは、CDがほとんど黒っぽい色だからだ。

タイトルや歌詞のリンクがクリックされると、モーダルが開いて日本語とローマ字の歌詞が表示されます。 日本語が読めない人でも、歌詞を読みながら歌えるようにローマ字表記を追加しました。 これにより、より国際化できるようになります。 歌詞はJSONファイルでHTMLファイルに含まれている。

また、<a>タグの中に配置すると、ストリーミングなどの他のサービスや、アルバムや曲を購入できる場所にリンクできるSVG画像もいくつか作成した。

index.html
<script>
const jsonText = document.getElementById('songData').textContent;
const songs = JSON.parse(jsonText);

function makeLyrics(track) {
    const selectedSong = songs.song[track-1];
    document.getElementById('trackmodal').textContent = selectedSong.title;
    document.getElementById('trackjapanese-pane').innerHTML = selectedSong.japanese;
    document.getElementById('trackromaji-pane').innerHTML = selectedSong.romaji;
}
</script>
